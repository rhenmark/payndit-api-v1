

## Description

Payndit - a platform designed to connect you with the right service providers in your community, quickly and easily.

Visit us here [Payndit](https://payndit.vercel.app/about)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
